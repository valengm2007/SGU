# ROLES

# personal administrativo:
- puede establecer horarios de inscripción
- puede habilitar o inhabilitar el sistema
- puede subir noticias y avisos
- puede gestionar parámetros del sistema

# docente:
- puede crear evaluaciones
- puede consultar las comisiones
- puede cargar notas
- puede cargar inasistencias

# alumno:
- puede preinscribirse a la facultad (generar usuario)
- puede inscribirse a materias 
- puede inscribirse a parciales
- puede consultar reportes académicos
- puede actualizar datos personales
- puede solicitar certificados

# alumno de extensión (próximamente):
- puede inscribirse a cursos
- puede realizar y gestionar preinscripciones a propuestas
- puede consultar reportes de los cursos
- puede editar sus datos
